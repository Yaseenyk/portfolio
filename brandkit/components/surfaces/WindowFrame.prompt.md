The "mission control" desktop-window shell — macOS traffic lights, centered mono filename, glowing LIVE status — used to house major page content.

```jsx
<WindowFrame title="portfolio -- interactive-mode" status="live">
  <div style={{ padding: "3rem" }}>…sections…</div>
</WindowFrame>
```

Title format is mono with ` -- ` separators. Pad the body yourself (site uses 2–3rem).
