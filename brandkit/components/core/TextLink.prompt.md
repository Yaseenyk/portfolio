Inline text link — zinc body color, cyan on hover, optional unicode arrow that nudges on hover.

```jsx
<TextLink href="/blog" arrow="right">Read Article</TextLink>
<TextLink href="https://github.com/Yaseenyk" arrow="diagonal">View source</TextLink>
```

`arrow="right"` (→ shifts right) for internal nav; `arrow="diagonal"` (↗ shifts up-right) for external links.
